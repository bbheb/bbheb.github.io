Change Hebrew Font for the Vocabluary Section on Canvas (Mac Only)

  • This tutorial demonstrates how to create an Apple Service to change the Hebrew font in the Matching Questions on Canvas.
    • Canvas does not allow us to change fonts for the matching questions (classical quizzes).
    • Using this setup, we only need to type a hotkey to make the font change.
  • Click this link for downloading the workflow file used in the tutorial.
  • Note: this method works for Mac users only. It works on Safari and all Chromium-based browsers. It does NOT work for Firefox, because it does not have "Run JavaScript from Apple Event" support.
  • If you are not a Mac user, the easiest way is to use the method shown on this page.
