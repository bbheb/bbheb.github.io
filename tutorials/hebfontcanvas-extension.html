Change Hebrew Font in Matching Questions on Canvas (Browser Extension)

  • This tutorial demonstrates how to install a (Chromium-based) browser extension to change the Hebrew font in the Matching Questions on Canvas.
    • Canvas does not allow us to change fonts for the matching questions.
    • Using this setup, it will automatically change the font.
  • Click this link for downloading the extension file used in the tutorial.
  • Note: this method works for Chromium-based browsers only, such as Google Chrome, Microsoft Edge, Brave, etc.
