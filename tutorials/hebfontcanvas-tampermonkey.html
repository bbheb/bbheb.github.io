Change Hebrew Font in Matching Questions on Canvas (Tempermonkey Extension)

  • This tutorial shows how to use the Tempermonkey browser extension to change the Hebrew font in the Matching Questions on Canvas.
    • Canvas does not allow us to change fonts for the matching questions.
    • Using this setup, it will automatically change the font.
  • Steps
    1. Install the Tempermonkey browser extension.
      • Go to this page, click the corresponding tab to your browser, and then install the extension.
        • Brave, Vivaldi, etc. are Chromium-based and should install from the Chrome store.
        • The Safari extension is priced at $1.99, all other browser extensions are free. Safari is not recommended for the course anyway.
    2. After installing the Tempermonkey extension to your browser, click this link and install the JavaScript.
  • Done!
