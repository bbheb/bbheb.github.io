Install the Hebrew (SIL) Keyboard

  • Install the Hebrew (SIL) keyboard on Windows.
  • Install the Hebrew (SIL) keyboard on MacOS.
    • Please visit the SBL official site to download the keyboard installation file as you follow the above tutorials.
  • Install the Hebrew (SIL) keyboard on iOS and android devices.

Hebrew Typing Test

  • Please visit the Hebrew Typing Test page.
  • This is also the best place to check your answer. When you see your answer look identical to the answer key but is marked as incorrect by Canvas, you may copy and paste your answer as well as the answer key to the input box. Then you will be able to see the differences between the two.
